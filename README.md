## Cách chạy dự án

Dự án hỗ trợ 2 cách chạy.

### Cách 1: Docker Compose (chạy toàn bộ trong container)

```bash
cd lawRag
docker compose up --build
```

Dừng hệ thống:

```bash
Ctrl + C
```

---

### Cách 2: Chạy local với virtual environment

(Chỉ Qdrant chạy bằng Docker)

```powershell
cd lawRag
.\start.ps1
```

Dừng hệ thống:

```powershell
Ctrl + C
```
